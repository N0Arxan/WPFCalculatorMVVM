Detailed Description

A utility class for interacting with the Windows Desktop Window Manager (DWM) API. Provides functionality to modify window attributes, such as enabling dark mode for the title bar.

Sets the value of a Desktop Window Manager (DWM) non-client rendering attribute for a window. This is a P/Invoke declaration for the native dwmapi.dll function.

Parameters
hwndA handle to the window for which the attribute value is to be set.
dwAttributeA flag describing which value to set.
pvAttributeA pointer to an object containing the attribute value.
cbAttributeThe size, in bytes, of the attribute value.
Returns
An HRESULT success or error code.
